Технічне завдання

Загальне по квартирі

  • Повний демонтаж + вивіз сміття та старих меблів
  • Повна заміна електропроводки (прокладка по підлозі), нові лінії, додаткові точки, заміна щитка
  • Демонтаж старих + встановлення 6 дверей (кухня, туалет, ванна, 2 кімнати, внутрішні вхідні)
  • Вирівнювання стін ПІД 90° (там, де будуть меблі)
  • Прокладання прихованого UTP для Wi-Fi
  • Фарбування стін: стримані тони, візуальне розширення простору

🍴 Кухня

  • Демонтаж старої плитки зі збереженням (фартух)
  • Фарбування 3 стін «під бетон»/рельєфна текстура
  • Підлога — ламінат
  • Заміна раковини, сифона
  • Підключення газової плити, монтаж витяжки

🚽 Туалет

  • Ревізія/повна заміна труб
  • Демонтаж старого + монтаж нового бойлера
  • Підлога — плитка
  • Вологостійка фарба «під бетон»

🛁 Ванна

  • Повна ревізія труб, заміна умивальника
  • Підключення пральної машини
  • Ванна зашивається з люком доступу
  • Плитка на підлозі + стіни по периметру ванни
  • Тепла підлога електрична

🚪 Коридор + кімнати

  • Стіни: рельєфна текстура «під бетон»
  • В коридорі — миюча стійка фарба
  • Підлога — ламінат (кімнати + коридор)
Стелі — обговорюється окремо після замірів. Додаткові роботи — після погодження.
